This page connects to a Neural Network which has been trained to distinguish between benign and malignant breast cancer cells based on a set of 9 measured features. The classifier predicts class '0' for benign and class '1' for malignant.

Five cell samples have been selected at random for you to test the network with. As a user, you can clearly see which cells are benign and maligant. When the "Test Cell" button is clicked, the Neural Network is fed the 9 measurement features of the cell, but it is NOT told which class the cell belongs to. The network will then return a prediction for which class the cell belongs to (Benign or Malignant).

Note: None of these randomly selected cell records have been shown to the neural network before, which demonstrates the network's ability to create a generalized solution to this classifcation problem. Accuracy for the model is at ~98% based on ~700 training records.
